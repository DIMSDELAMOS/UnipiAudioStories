**An Interactive Android Storytelling App for Children** 

Unipi Audio Stories is an Android application designed to entertain children by narrating classic fairy tales using built-in Text-To-Speech (TTS) technology. 

## ‚ú® Key Features

* **Interactive Story Library:** A visual list of classic tales displaying the title, author, and cover image.
* **Text-To-Speech Narration:** Uses Android's native TTS to read the stories out loud (currently optimized for Greek). Users can easily start and stop the narration.
* **Multilingual UI:** The application interface dynamically adapts to Greek, English, and French based on user selection.
* **User Statistics:** A dedicated dashboard tracks the total number of stories listened to and calculates the user's "Favorite Story".
* **Tablet Optimized:** The UI is fully responsive and scales beautifully for larger Android tablet screens.
